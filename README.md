## What it does

| Feature | Description |
|---|---|
| 🤖 **AI Investment Advisor** | 5-question onboarding flow → weighted risk profiling → personalized fund recommendations + live SIP calculator |
| 🎯 **Goal Planner** | 6 life goals (education, retirement, home, wedding, travel, emergency) with dynamic SIP calculation using FV formula |
| ⚖️ **Fund Comparator** | Side-by-side comparison of 8 real Indian mutual funds with verified 1Y/3Y/5Y CAGR data, expense ratios, AUM |
| 📊 **Market Insights** | Live market context, top performers, sourced news, and product thinking breakdown |

---

## Product Thinking Highlights

- **Funnel Design** — 5-question onboarding minimizes drop-off while maximizing profiling signal quality
- **Personalization Engine** — 3-signal weighted risk model (age × risk tolerance × investment horizon)
- **Goal-Based Journeys** — SIP calculator uses Future Value formula accounting for existing savings
- **Data Integrity** — Fund returns verified from Groww, Value Research, AMC sites (as of June 2026)
